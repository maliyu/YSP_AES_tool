AES-CBC tool. .Net Framework 4 or higher version is required for this tool!

It is developed by Yosun Singapore. 

Input plaintext, key and IV, then click <Encryption>/<Decryption> button. The ciphertext should be shown on Ciphertext textbox. 

User can compare the result with expected resuly by clicking <Compare> button. Don't forget to input expected result into Comparison Ciphertext textbox.

User can get some help by clicking <Help> button.
